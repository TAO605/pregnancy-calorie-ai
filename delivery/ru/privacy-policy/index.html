Политика конфиденциальности

Последнее обновление: 13.05.2026

«AI Pregnancy Calorie Calculator» — это персональный инструмент SaaS, управляемый Тао Сю. В настоящей политике объясняется, какая информация необходима для расчета суточной нормы калорий во время беременности, предоставления рекомендаций по питанию на основе искусственного интеллекта, доступа к учетной записи и оформления дополнительных премиум-подписок.

Информация, которую мы собираем

Мы собираем только ту информацию, которая необходима для работы данного сервиса:

  • Данные учетной записи: ваш адрес электронной почты и сеанс входа, защищенный паролем, при создании учетной записи.
  • Данные калькулятора срока беременности: возраст, рост, вес до беременности, срок беременности, тип беременности, уровень активности, предпочтительные единицы измерения, дополнительные предпочтения в питании, аллергии, бюджет, уровень сложности приготовления блюд, а также еженедельные записи о весе, которые вы решите ввести.
  • Данные запроса ИИ: результат расчета и ваш вопрос о питании во время беременности или запрос на индивидуальную настройку, чтобы система искусственного интеллекта могла сформировать рекомендации.
  • Данные об оплате: статус подписки, тип тарифного плана, идентификатор клиента Stripe, данные о продлении или истечении срока действия, а также идентификаторы платежных операций, необходимые для управления доступом к премиум-версии.
  • Технические характеристики: стандартные журналы хостинга, такие как IP-адрес, тип браузера, запрашиваемый URL, дата, время и диагностические события, используемые для обеспечения безопасности и надежности.

Обработка платежей через Stripe

Обработка платежей осуществляется сервисом Stripe. Мы не храним, не обрабатываем и не имеем доступа к полному номеру вашей кредитной карты, коду CVC или данным для аутентификации карты. Сервис Stripe может собирать платежные реквизиты и информацию о способе оплаты в соответствии со своими собственными политиками конфиденциальности и безопасности.

Как мы используем информацию

  • Для расчета суточной нормы калорий при беременности, целевых показателей по макроэлементам, ориентировочных значений прибавки в весе и сводных данных о питании.
  • Создать основанные на искусственном интеллекте рекомендации по питанию во время беременности на основе предоставленной вами информации.
  • Для создания и управления учетной записью, сессией входа, сброса пароля, статусом подписки и доступом к премиум-функциям.
  • Для оказания поддержки клиентам при обращении к нам.
  • Для защиты веб-сайта, предотвращения злоупотреблений, устранения ошибок и повышения надежности сервиса.

Локальное хранилище браузера

Сайт может сохранять ваши последние вводные данные в калькуляторе и записи еженедельного отчета в локальном хранилище вашего браузера. Это позволяет вам вернуться к калькулятору, не вводя все данные заново. Вы можете удалить эти данные, очистив данные сайта в настройках браузера.

Обмен информацией с поставщиками услуг

Мы прибегаем к услугам сторонних поставщиков только в тех случаях, когда это необходимо для работы продукта: Vercel — для хостинга и бессерверных функций, поставщик услуг по управлению базами данных — для хранения учетных записей и информации о статусе подписок, Stripe — для обработки платежей, Resend — для рассылки писем по сбросу паролей, а также поставщики API искусственного интеллекта — для обработки запросов на получение рекомендаций по питанию. Мы не продаем персональные данные.

Информация о здоровье и беременности

Введенные вами данные о беременности используются для расчета ориентировочных значений по питанию и составления рекомендаций по искусственному вскармливанию. Они не используются для постановки медицинских диагнозов, лечения, принятия решений по страхованию или создания рекламных профилей.

Хранение данных

Данные об учетной записи и подписке хранятся до тех пор, пока ваша учетная запись остается активной, а также в случае необходимости для обеспечения безопасности, в соответствии с требованиями законодательства, для целей бухгалтерского учета или в случае возникновения споров. Данные локального хранилища браузера остаются на вашем устройстве до тех пор, пока вы их не удалите. Токены сброса пароля теряют силу и аннулируются после использования.

Ваши возможности и права

Вы можете пользоваться бесплатным калькулятором без оформления подписки. Вы можете очистить кэш браузера, отменить подписку через Stripe или связаться с нами, чтобы запросить доступ, исправление или удаление данных, связанных с вашей учетной записью, если это разрешено действующим законодательством.

Медицинское предупреждение

Данная услуга предназначена исключительно для просвещения в области питания. Она не предусматривает предоставления медицинских рекомендаций и не заменяет консультаций акушера, врача, дипломированного диетолога, акушерки или другого квалифицированного медицинского работника.
