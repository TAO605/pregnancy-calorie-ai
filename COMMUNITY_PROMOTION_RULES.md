# Community Promotion Notes

These notes are for compliant, low-pressure promotion of AI Pregnancy Calorie Calculator in overseas pregnancy communities.

## Universal Rules

- Help first, link last.
- Do not post the same template repeatedly.
- Do not reply to every thread with a link.
- Do not make medical claims, diagnosis claims, or guaranteed outcome claims.
- Always say the tool is for general nutrition guidance only.
- If you own, built, or are connected with the tool, disclose that relationship.
- Do not use private messages for promotion unless the user explicitly asks.
- Avoid weight-shaming language. Use supportive wording.
- If a user mentions gestational diabetes, hypertension, eating disorder history, severe nausea, rapid weight change, or twins/multiples, suggest discussing it with their healthcare provider.

## Quora

What works:
- Clear, complete answers that directly answer the question.
- One relevant link after helpful context.
- Transparent disclosure if connected to the linked tool.

Avoid:
- Link-only answers.
- Repeating the same answer across many questions.
- Hiding affiliation.
- Overly promotional language.

Suggested frequency:
- 2-3 helpful answers per week.
- Add a link only when the question directly relates to pregnancy calories, nutrition planning, or weight gain.

## Reddit r/pregnancy

What works:
- Casual, empathetic comments.
- Sharing personal experience or practical advice.
- Very low-frequency link mentions when the original poster asks for a tool.

Avoid:
- New account posting links immediately.
- Copy-paste replies.
- Promotional titles or sales language.
- Medical advice, diagnosis, or "you should do X" wording.

Suggested frequency:
- Build account history first.
- 2-3 helpful comments per week.
- Link only when relevant and disclose connection.

## BabyCenter

What works:
- Warm, supportive replies.
- Practical food ideas and reassurance.
- Encouraging users to speak with their provider for personal medical concerns.

Avoid:
- Commercial-looking posts.
- Aggressive tool promotion.
- Claims that a calculator can replace prenatal care.
- Repeating the same link across multiple discussions.

Suggested frequency:
- 1-2 supportive replies per week.
- Mention the tool only when the thread is clearly about calories, nutrition planning, or weight gain.

## What to Expect

What works:
- Supportive, mom-to-mom style replies.
- Short, practical guidance.
- Low-pressure tool mention at the end.

Avoid:
- Posting as if you are a doctor.
- Promising a correct number for every pregnancy.
- Using fear-based language around weight gain.
- Posting links in unrelated symptom or diagnosis threads.

Suggested frequency:
- 1-2 replies per week.
- Use the disclosure line whenever mentioning the tool.

## Safe Disclosure Lines

Use one of these when appropriate:

- "Disclosure: I am connected with this tool, so please treat it as a general planning resource, not medical advice."
- "Disclosure: I helped build this calculator. It is meant to be a starting point for nutrition planning, not a replacement for your provider."
- "Disclosure: This is my project, so no pressure to use it. I am sharing it only because it fits your question."

## Phrases to Avoid

- "This will tell you exactly what you need."
- "This is better than asking a doctor."
- "Guaranteed healthy pregnancy."
- "You must eat..."
- "You are gaining too much."
- "You are doing it wrong."
- "Cure," "diagnose," "treat," or medication-related advice.
