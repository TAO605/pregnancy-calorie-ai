# Lessons Learned

## What Makes a Safe Community Reply

- Answer the user's question first.
- Keep the tool mention short and optional.
- Add disclosure if connected with the tool.
- Use supportive, non-judgmental language.
- Avoid strict commands and medical claims.

## Best-Fit Topics

The tool is most appropriate when the thread is about:

- Daily pregnancy calorie needs.
- Trimester calorie changes.
- Pregnancy weight gain ranges.
- Meal planning during pregnancy.
- Finding a pregnancy calorie calculator.

The tool is not appropriate when the thread is about:

- Emergency symptoms.
- Medication decisions.
- Medical diagnosis.
- Eating disorder distress.
- High-risk pregnancy management.

## Best Posting Workflow

1. Read the full thread.
2. Confirm the question is directly relevant to pregnancy nutrition or calorie planning.
3. Write a custom first sentence that references the user's situation.
4. Use one template as a base.
5. Add disclosure.
6. Post no more than a few helpful replies per week per community.

## Reuse Rule

Do not copy the same template word-for-word. Change the opening, examples, and closing based on the thread context.
