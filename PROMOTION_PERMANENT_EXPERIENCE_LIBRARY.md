# Permanent Promotion Experience Library

This library summarizes reusable rules from the preparation workflow for AI Pregnancy Calorie Calculator.

## 1. Data Truthfulness

- Never invent keyword volume, KD, DR, traffic, backlinks, or competitor metrics.
- If a metric is unavailable, write `数据不可得` or `unavailable`.
- Ahrefs DR is not the same as Semrush Authority Score.
- KD unavailable is not equal to low competition.
- Free tool limits must be recorded instead of bypassed with assumptions.

## 2. Keyword Research

- Use real SEO tools for keyword volume and KD.
- Query each target country separately if the tool only supports one database at a time.
- Dedupe by normalized keyword.
- Sort numeric KD first; put unavailable KD after verified numeric rows.
- Keep KD-unavailable keywords as a future validation pool.
- Do not generate blog titles, outlines, meta descriptions, or content snippets from keyword data.

## 3. Competitor Analysis

- Use the actual Google result page and independent domains.
- Do not count multiple pages from the same domain as separate competitors.
- If a page is blocked or unavailable, mark it instead of guessing.
- Separate page observations from domain-level SEO metrics.
- Strong competitors may have high domain authority but weak product depth.
- Weak competitors can be beaten with better trust, UX, personalization, and conversion.

## 4. Backlink Building

- Separate verified DR≥80 resources from candidate resources.
- Start with brand entity profiles before community links.
- Use canonical website URLs.
- Keep brand name, description, and support email consistent.
- Do not spam communities or use private messages for promotion.
- Update the progress table after each link is created or submitted.

## 5. Community Promotion

- Help first, link last.
- Disclose affiliation.
- Customize every reply.
- Do not post the same template repeatedly.
- Use low posting frequency.
- Avoid medical advice, diagnosis, treatment, or guaranteed outcomes.
- When in doubt, suggest talking with a healthcare provider.

## 6. YMYL Health Safety

- Position the product as general nutrition guidance.
- Mention that it does not replace doctors, midwives, registered dietitians, or prenatal care.
- Avoid fear-based weight language.
- Avoid words like cure, diagnose, treat, guarantee, must, or forbidden.
- Use supportive language around pregnancy weight and nutrition.

## 7. Tracking and Iteration

- Track GSC impressions before expecting clicks.
- Track pricing views before expecting subscriptions.
- Use UTMs for every external promotion channel.
- Review metrics weekly, not hourly.
- Build new pages only from real query data or validated keyword data.
- Keep issue logs updated when data is blocked or unavailable.

## 8. Operating Rule for Future Tasks

Before any new promotion task, read:

- `PROMOTION_MASTER_ISSUE_LOG.md`
- `PROMOTION_PERMANENT_EXPERIENCE_LIBRARY.md`
- `PROMOTION_EXPERIENCE_LIBRARY_USAGE.md`
- The latest task-specific output file.

Then state which rules apply before executing.
